### Setup

* .env:  set password, host , dbname
```
DATABASE_URL="postgresql://postgres:__PASS__@__HOST__.supabase.co:5432/dbname"

POSTGRES_DATABASE=""
POSTGRES_USER="postgres"
POSTGRES_PASSWORD=""
POSTGRES_HOST="db._HOST.supabase.co"
POSTGRES_PORT=5432
```

* npm add
```
npm install
```


* start


```
# dev-start
yarn dev

# build
yarn build

# start
yarn start
```
